Know Your Gov

Description

Know Your Gov is a civic informational tool where users can look up their elected representatives at the state and federal levels using an address. The full stack application utilizes the Google Civic Information API to deliver the elected representatives contact information. The responsive mobile first design allows users to easily contact their representative via phone, email, or through their website. Users also have the option of registering for the service and storing their address in the database to save their representatives information for future reference. Know Your Gov is a great way of getting in touch with your representatives!

The Community Toolbox

Description

The Community Toolbox is an online tool library where users can find and rent tools for home improvement projects. The full stack application allows the user to create a profile for reserving tools and checking them out. When a tool is checked out the user is given instructions to the warehouse and how the pick-up process works. The user interface allows the user to quickly check a tools availability and to keep track of their checked out tools. The Community Toolbox is an excellent reservation resource for local tool libraries to feature their inventories online.

TripSpiration

Description

The TripSpiration application relies on four APIs (FourSquare, OpenWeatherMap, Pixabay, Dark Sky) to inspire the user to travel to a particular city. The content is dynamically generated based on user input. When prompted to enter a travel destination they're curious about and their preferred travel month, the application returns a gallery of spectacular vistas of the city, current and projected weather conditions based on historical weather patterns, in addition to a list of popular local points of interest to visit. TripSpiration is here to help inspire your next trip!

Mac OS HotKey Quiz

Description

The Mac HotKey Quiz app is built for the Mac user who wants to test their HotKey knowledge and learn a few new tricks. The quiz runs through the most important HotKeys for operating your Mac computer. At the end of the quiz, the user receives their score and determines whether or not they need to study more.
